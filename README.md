# csv_finder
output csv entries with keyword

### Note: first line of csv is used to get the index of keyword to parse the rest!

# Usage
Compile:  
```script
make
```
  
Run:  
```script
./cf csv_file_address keyword_to_find
```
  
Clean:  
```script
make clean
```
